# AutoDocGen

AutoDocGen is a Python tool that generates documentation for a Python project automatically. It parses docstrings and comments to create structured HTML documentation.

## Features

- **Automatic Documentation**: Generates documentation from docstrings and comments.
- **HTML Output**: Creates structured HTML documentation.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/wasserpuncher/AutoDocGen.git
    cd AutoDocGen
    ```

2. Create and activate a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

3. Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. Run the documentation generator:
    ```bash
    python doc_generator.py
    ```

2. The script will generate HTML documentation in the `docs` directory.
